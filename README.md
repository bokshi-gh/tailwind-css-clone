# Tailwind CSS Clone

A lightweight, modular **Tailwind CSS clone** built from scratch.

## Getting Started

1. Link via `<link>`
```html
<link rel="stylesheet" href="https://raw.githubusercontent.com/bokshi-gh/tailwind-css-clone/main.css">
```

2. Link via CSS `@import`
```css
@import url("https://raw.githubusercontent.com/bokshi-gh/tailwind-css-clone/main.css");
```

3. Clone repository and add locally
