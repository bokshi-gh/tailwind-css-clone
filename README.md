# Tailwind CSS Clone

A lightweight, modular **[Tailwind CSS](https://tailwindcss.com/) clone** built from scratch.

## Getting Started

1. Link via `<link>`
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/bokshi-gh/tailwind-css-clone/main.css">
```

2. Link via CSS `@import`
```css
@import url("https://cdn.jsdelivr.net/gh/bokshi-gh/tailwind-css-clone/main.css");
```

3. Clone repository and add locally
